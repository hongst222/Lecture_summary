트랜지션

transition-duration(필수)

애니메이션이 진행되는 시간

시간 단위는 초(seconds=s) 또는 밀리초(milliseconds=ms=1/1000초)
1초 아래의 숫자는 0을 빼고 입력해도 된다.

0.5초
2초

transition-property

원하는 트랜지션 효과를 따로 줄 수 있다.

transition-property: all; 요소의 모든 속성에 트랜지션 지정(기본값)
transition-property: color; 글자색에만 트랜지션 지정
transition-property: color, border-color; 글자색, 테두리색상만 트랜지션 지정
transition-property: none; 트랜지션 해제

transition-delay

트랜지션 지연 시간 설정(이벤트 발생 후 기다리는 시간)

0.5초
2초

transition-timing-function

트랜지션이 진행되는 속도(변화가 일어나는 속도)를 조절

ease : 시작은 느리게, 빨라졌다가 종료지점에서 다시 느려짐(기본값)
ease-in : 시작을 느리게
ease-out : 종료를 느리게
ease-in-out : 시작지점과 종료지점의 변화가 천천히 진행
linear : 시작부터 종료까지 동일한 속도로 진행
cubic-bezier : 처음과 끝의 속도를 0과 1 사이의 수치를 이용해 함수값을 직접 지정(http://cubic-bezier.com)

ease-in

transition속성 축약형

transition: property duration timing-function delay;(순서 상관X, duration과 delay만 순서대로. 한개의 초단위값만 입력할 경우 duration으로 인식)

생략된 값은 기본값으로 적용. 보통 property와 duration을 많이 사용합니다.
기본값 : all 0 ease 0;
